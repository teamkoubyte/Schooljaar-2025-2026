📘 Wiskunde Samenvatting – Logaritmen & Exponentiële Functies

Deze samenvatting bevat de belangrijkste theorie over logaritmen, exponentiële functies en groeimodellen.

1. Verbanden tussen grootheden

In wiskunde bestuderen we vaak een verband tussen twee variabelen.

Voorbeelden:

tijd en afstand

tijd en aantal bacteriën

tijd en geld

Een verband kan worden voorgesteld met:

een tabel

een grafiek

een formule

2. Soorten verbanden
2.1 Lineair verband

Formule:

[
y = b + ax
]

Betekenis:

(b) = beginwaarde

(a) = groeiterm

Kenmerken:

de toename is constant

de grafiek is een rechte

Voorbeeld:

week	geld
0	10
1	15
2	20
3	25

Formule:

[
y = 10 + 5x
]

2.2 Omgekeerd evenredig verband

Formule:

[
y = \frac{a}{x}
]

Kenmerken:

als (x) verdubbelt → (y) halveert

grafiek = hyperbool

2.3 Kwadratisch verband

Formule:

[
y = ax^2 + bx + c
]

Kenmerken:

toename van de toename is constant

grafiek = parabool

3. Exponentiële groei

Formule:

[
y = b \cdot a^x
]

Betekenis:

(b) = beginwaarde

(a) = groeifactor

(x) = tijd

Kenmerken:

er wordt vermenigvuldigd met een factor

grafiek = kromme

Voorbeeld:

[
N = 100 \cdot 2^t
]

4. Groeifactor en groeipercentage

Formule:

[
a = 1 + \frac{p}{100}
]

Voorbeelden:

groei	groeifactor
3%	1.03
5%	1.05
-2%	0.98
5. Exponentiële afname

Wanneer iets daalt met een percentage.

Voorbeeld: 5% verlies per dag.

[
a = 1 - 0.05 = 0.95
]

Formule:

[
y = b \cdot 0.95^x
]

6. Exponentiële vergelijkingen

Voorbeeld:

[
2^x = 16
]

Omdat:

[
2^4 = 16
]

Dus:

[
x = 4
]

Wanneer dit niet direct zichtbaar is, gebruiken we logaritmen.

7. Logaritmen

Logaritmen zijn het omgekeerde van exponenten.

Definitie:

[
\log_a x = y
]

betekent:

[
a^y = x
]

Voorbeeld:

[
\log_2 8 = 3
]

want

[
2^3 = 8
]

8. Briggse logaritmen

Logaritmen met grondtal 10.

[
\log x = \log_{10} x
]

Voorbeelden:

[
\log 100 = 2
]

[
10^2 = 100
]

[
\log 0.1 = -1
]

9. Belangrijke eigenschap

Je kan geen logaritme nemen van een negatief getal.

[
\log(-6)
]

→ geen oplossing

10. Rekenregels van logaritmen
Productregel

[
\log(ab) = \log a + \log b
]

Voorbeeld:

[
\log 3 + \log 5 = \log 15
]

Quotiëntregel

[
\log\left(\frac{a}{b}\right) = \log a - \log b
]

Voorbeeld:

[
\log 24 - \log 4 = \log 6
]

Machtsregel

[
\log(a^k) = k \log a
]

Voorbeeld:

[
\log(5^3) = 3\log 5
]

Algemene formule:

[
N = N_0 \cdot a^t
]

Voorbeeld:

[
500 \cdot 1.02^t = 1000
]

Bijvoorbeeld bij geluidssterkte (decibel).

[
L = 10 \log\left(\frac{I}{I_0}\right)
]
